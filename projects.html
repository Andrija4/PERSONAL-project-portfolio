Web Service for tracking equipment calibration dates
Side project assigned to me at Brose, built to track calibration dates of various equipment used in the lab. The web service is built using FastAPI for the backend, Jinja2 for templating, and HTMX for dynamic interactions, allowing users to easily view, add, and manage calibration records through a user-friendly interface.

A custom-built RFID door access system designed for a university laboratory, utilizing an ESP32 microcontroller and RC522 RFID reader. The system allows authorized personnel to gain entry by scanning their RFID cards, with access logs stored in a FastAPI backend for monitoring and management.
